Getting Started
===============

For initial orientation, a :ref:`quick overview <repository_contents>`
of the main contents of the `LFRic Apps <lfric_apps_github_>`__ repository is
given. As noted, the `LFRic Core <lfric_core_github_>`__ is developed in
separate repository.

This repository contains all the main science code developed for the LFRic
Atmosphere model as well as a number of applications for both running atmosphere
simulations and testing various aspects of the science code.
